# 🐎 Injekt - Network & Vulnerability Scanner

> A lightweight Python-based Network & Vulnerability Scanner built for cybersecurity learning and defensive testing.

---

## 📌 Overview

**Injekt** is a terminal-based network scanning and vulnerability assessment tool written in Python.  
It is designed for students, cybersecurity enthusiasts, and ethical hackers who want to understand how network reconnaissance works.

The tool combines:

- 🔍 Network discovery
- 🚪 Port scanning
- 🧠 System information gathering
- 🎨 Clean CLI interface with colored output
- ⚡ Lightweight & fast execution

---

## 🚀 Features

- 📡 Network host discovery
- 🔎 Open port detection
- 🖥 OS and system information display
- 📊 Clean ASCII banner interface
- 🎨 Colorized terminal output using `colorama`
- 📦 Automatic dependency installation
- 🧠 Lightweight Python architecture

---

## 🛠 Built With

- **Python 3.x**
- `nmap`
- `colorama`
- `psutil`
- `subprocess`
- `platform`
